JSSC — JavaScript String Compressor 2.0.0 ​

JSSC (JavaScript String Compressor) - алгоритм сжатия текста без потерь с открытым исходным кодом, разработанный специально для JavaScript.

Он работает непосредственно с string в JavaScript (UTF-16) и создает сжатые данные, которые также являются допустимыми в JavaScript string.

JSSC является UMD модулем и может использоваться в браузерах, Node.js, Deno и других средах JavaScript.

Ключевые Возможности ​

  • ✨ Сжатие без потерь
  • 🗜️ Высокая степень сжатия
    • до 8:1 для чисел
    • отличные результаты при работе с повторяющимся и структурированным текстом
  • 🌍 Многоязычность
    • Английский, Русский, Японский, Китайский, Хинди, Бенгальский, и другие
  • 📦 Поддержка JSON
    • JSON конвертируется в JUSTC перед сжатием
  • ⚙️ String → String
    • нет двоичных буферов
    • нет внешних метаданных
    • вся необходимая информация встроена в саму сжатую string
  • 🧠 Самоподтверждающееся сжатие
    • прежде чем быть кандидатом, каждый режим сжатия проверяется расжатием
  • 🔁 Рекурсивное сжатие
  • 📜 Поддержка TypeScript
  • 🌐 Поддержка браузеров и статических веб-сайтов

Важное Замечание О Совместимости Версий ​

⚠️ Сжатые строки, созданные JSSC версии 1.x.x, несовместимы с версией 2.x.x

Причины:

  • Первые 16 бит (расположение заголовка) были немного переработаны
  • Добавлены новые режимы сжатия
  • Расширены таблицы кодировок символов

Кодировка Символов ​

JSSC работает c кодовыми единицами JavaScript UTF-16, а не байтами UTF-8.

Это означает, что:

  • Поддерживается любой символ, который может быть представлен в строке JavaScript
  • Сжатие выполняется на уровне UTF-16
  • Один символ JavaScript = 16 бит
  • Двоичные данные должны быть преобразованы в строки перед сжатием

Название Проекта И Название Пакета ​

Проект называется JSSC (JavaScript String Compressor).

Пакет npm опубликован под именем strc (сокращение String Compressor), поскольку имя jssc уже занято в npm другим пакетом на базе Java, который не имеет никакого отношения к данному проекту.

Оба названия относятся к одному и тому же проекту.

Установка ​

Через npm

bash
npm i strc

Название пакета — strc, но сама библиотека называется JSSC.

Или можно использовать прямо на статическом веб-сайте вставив эти HTML script тэги.

html
<script src="https://unpkg.com/justc"></script>
<script src="https://unpkg.com/strc"></script>

API ​

compress(input: string | object | number): Promise<string> ​

Сжимает инпут и выдаёт сжатую JavaScript string.

decompress(input: string, stringify?: boolean): Promise<string | object | number> ​

Расжимает ранее сжатую string или объект или целое число.
